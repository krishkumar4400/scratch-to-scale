2. Understanding the req (request) object.

The req object contains information about the http request made by the client.

Common Properties of req:
    - req.method: The HTTP method (e.g, GET, POST)
    - req.url: The requested URL (e.g, /home)
    - req.headers: The headers sent by the client. 
    - req.on('data'): for receiving the request body in chunks.
    - req.on('end'): Indicates the end of the request body.

3. Understanding the res(Response) Object:

The res object is used to send a response back to the client.

common Methods of res:
    - res.writeHead(statusCode,headers): Set the statuscode and headrs 
    - res.write(data): Write data to the response body.
    - res.end(): End the response.
